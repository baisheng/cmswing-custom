只在项目为可选时有效，每行一个字段，等号前面为字段索引(建议用数字)，后面为内容，例如: 
1:选项1
2:选项2
3:选项3
1.1:选项1-1
1.2:选项1-2
1.2.1:选项1-2-1
注意:
1、 "1.2.1:选项1-2-1"必须有"1.2 = 选项 1-2"和"1 = 选项1"这两项
2、 "1.2.1"之间不能有空格
3、 字段确定后请勿修改索引和内容的对应关系，但仍可以新增字段。如需调换显示顺序，可以通过移动整行的上下位置来实现
